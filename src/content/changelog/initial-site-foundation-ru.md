Публичный сайт ушел от одной длинной страницы к более понятной структуре для buyers.

## Добавлено

- компактная главная страница с фокусом на workflow и handoff менеджеру
- отдельные product-страницы для более глубокого операционного описания
- маршруты blog и changelog с локализованными коллекциями
- локализованные пути на английском, русском и арабском для каждой опубликованной записи

## Улучшено

- навигация в хедере и футере для SEO-маршрутов второго уровня
- canonical и alternate metadata для контентных страниц
- публичная структура release notes для следующих обновлений сайта

## Зачем это нужно

Потенциальный клиент должен быстро считать главную страницу, а затем перейти к более глубоким материалам только при необходимости. Этот релиз делает такой маршрут понятнее и для читателя, и для поисковой индексации.
